Measurements in CSS

There are 4 main ways for measurements

  • Pixels: In CSS we would use px

    • Example: padding: 10px; //which is 10 pixels away from a text

  • Percentage: We would use % symbol to indicate the size based of the screen

    • Example: padding: 10%; //which is 10 percent of the screen

  • Em: Em is used mainly for fonts and for others objects within a document

    • Example: If you wanted to have a font of size 12px, you would need to convert it to em by dividing the size by 16.

    Note: It would scale pased off the parent font.

  • Rem: rem is imilar to em, except it is the root of the font size.

    • Root ems, are always relative to the font-size of the html element

    Example: If the font sizing of a font is 12px, an object that has a rem measurment would not scale to that size.

CSS Layouts

In a HTML document, the positioning of each element is set to inline, which means that it would reposition itself and stay in line within each text or element.

To control the layouts of a page there are a couple of properties that are important to do so.

The Display Property

The display property specifies if/how and element is displayed.

The default display values are mostly block or inline.

Block elements always start on a new line and take up the full width available(stretches out to the left and right as far as it can)

Block level elements are:

  • div
  • h1-h6
  • p
  • form
  • header
  • footer
  • section

Inline elements usually do not start on a new line and only takes up as much width as necessary.

Inline elements are:

  • span
  • a
  • img

Width, Height, and Max-Width

When it comes to a responsive website, the width and height plays a key role.

  • Width determines how long an element is,
  • Height determines how long an image is,
  • Max-Width is the maximum length from right to left an element can be.

Height is the least of our problems, we can set the height to auto if there are large portions of text and can not have a specific height.

When dealing with smaller windows, the width becomes the main problem.

In this case, it's best to specify the max-width instead so that the width remains the same.
